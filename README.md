# 🌐 kelechi's website — Personal Portfolio



### This is the source code for my personal website. Built with React, Vite, and Tailwind CSS.
---

## 🛠️ Tech Stack

- React + Vite
- TypeScript
- Tailwind CSS
- shadcn/ui

---

## 🧑‍💻 Getting Started

### 📄 Clone the repo

```bash
git clone https://github.com/ukpabik/kelechi-website.git
cd kelechi-website
cd website
```

### 🧪 Install Dependencies
```bash
npm install
```

### 📦 Try it for yourself!
```bash
npm run dev
```
